# 📖 About

**BigFish-NX** is a carefully curated Nintendo Switch CFW package designed for users who want a clean, reliable and easy-to-maintain setup.

Rather than including every available tool, BigFish-NX focuses on quality, compatibility and simplicity. Every release is tested before publication to ensure the best experience possible.

Whether you're installing CFW for the first time or maintaining an existing Switch, BigFish-NX aims to make setup and updates straightforward.

# 🚀 Installation

## Fresh Installation

1. Download the latest Release.
2. Extract the archive.
3. Copy every file to the root of your SD Card.
4. Insert SD Card into your Nintendo Switch.
5. Launch Hekate.
6. Boot Atmosphère.
7. Enjoy.

---

## Updating

1. Backup your SD Card.
2. Download the latest release.
3. Replace the old files.
4. Boot normally.
